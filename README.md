# Frontend Intern Task

Full Stack Task Manager Application

## Tech Stack
- React (Vite)
- Node.js
- Express.js
- MongoDB
- JWT Authentication
- Tailwind CSS

## Features
- Register & Login
- JWT Authentication
- Create Task
- View Tasks
- Update Task
- Delete Task
- Search Tasks
- Protected Routes

## Backend Setup
cd backend  
npm install  
node server.js  

Create `.env` file in backend:

MONGO_URI=your_mongodb_connection_string  
JWT_SECRET=your_secret_key  

Backend runs on: http://localhost:5000

## Frontend Setup
cd frontend  
npm install  
npm run dev  

Frontend runs on: http://localhost:5173

## API Collection
Postman collection is included in the repository.
